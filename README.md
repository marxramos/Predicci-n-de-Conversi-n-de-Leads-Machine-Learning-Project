# Prediccion de Conversion de Leads Machine Learning Project

Este proyecto tiene como objetivo desarrollar un modelo de aprendizaje supervisado capaz de predecir si un lead que contacta a una empresa de alquiler de vehículos de alta gama se convertirá en una venta.
La empresa busca optimizar la asignación de esfuerzos comerciales y de marketing mediante un pronóstico anticipado de la calidad de sus leads. Un modelo de este tipo permite identificar qué contactos tienen mayor probabilidad de conversión y priorizar los recursos de forma más eficiente.
